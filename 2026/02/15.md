### わかったこと
- repeatメソッドを使用してループ処理をしないでも JavaScript のカレンダーで空白を出せること
もともとは
`for (let i = 0; i < dayOfWeek; i++) {
    firstRowSpaces += "   "
}`
と書いていたのだが、以下のようにrepeatメソッドを使うことで、ループ処理をしないでも、1行で空白が出せた
`let firstRowSpaces = "   ".repeat(currentDayOfWeek)`
- JavaScriptカレンダーでif文で数字を回していたが、padStartメソッドを使うことでif文が不要とのことだった。もともとは
` if (i < 10) {
   process.stdout.write(" " + i + " ");
   } else {
       process.stdout.write(i + " ");
   }`
と書いていたが、
`process.stdout.write(String(i).padStart(2, " ") + " ")`
とすることによって、一行のみでカレンダーの数値を出すことができた。
